# Google Sheets → Pipedrive CRM Lead Sync

Developed an automated workflow to sync new leads from Google Sheets to Pipedrive CRM in real time.
Mapped and validated lead fields (name, email, phone, source, notes) to ensure accurate data transfer.
Implemented duplicate checks to prevent redundant entries in the CRM.
Automated lead assignment and pipeline placement to streamline the sales process.
Configured automated error-handling with email notifications for failed or incomplete syncs.
Impact:

Eliminated manual CRM data entry
Improved lead response time and sales team efficiency
Increased data accuracy and pipeline visibility
Tech Stack: Google Sheets API • Pipedrive API • Automation Tools (Make.com/Zapier) • Webhooks • CRM Automation • Email Automation
